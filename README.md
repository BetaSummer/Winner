## 本地启动
- `cp config.sample.js config.js` 复制配置文件, 并做相应修改
- `npm install` 安装依赖
- 启动 `mongodb` 和 `redis`
- `gulp dist` 生成静态资源
- `npm run dev`  启动应用
- 浏览器访问 `127.0.0.1:3000`
